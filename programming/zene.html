Graphics

I have worked on the Zene Graphics library since September 2021, and is my biggest project so far. It is a library designed to fully manage OpenGL graphics from simple shapes to full 3D lighting. At the moment, there is a context based rendering system, that includes simple draw functions for rectangles, rounded boxes, circles and textures. There are also a range of default shaders included, for example: basic 3D lighting and depth map rendering for shadows.
